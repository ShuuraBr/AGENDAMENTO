Configuração necessária para os alertas financeiros

Defina uma das variáveis abaixo no ambiente da hospedagem:

- FINANCEIRO_EMAIL
- EMAIL_FINANCEIRO
- FINANCE_EMAIL
- SETOR_FINANCEIRO_EMAIL

Recomendado:

FINANCEIRO_EMAIL=financeiro@seudominio.com.br

Observações:
- Os alertas automáticos por ciência de vencimento e o resumo mensal usam esse destinatário.
- Se nenhuma dessas variáveis estiver definida, o sistema continua funcionando, mas os e-mails ao financeiro não serão enviados.
