# Deploy na Hostinger

## Configuração
- Framework preset: Express
- Root directory: backend
- Entry file: src/server.js
- Package manager: npm
- Node version: 20.x ou 22.x

## Variáveis
- PORT=3000
- FRONTEND_URL=https://agendamento.objetivaatacadista.com.br
- JWT_SECRET=troque_essa_chave

## Comandos
```bash
cd backend
npm install
npm start
```
