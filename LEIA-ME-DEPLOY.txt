CONFIGURACAO RECOMENDADA NO DEPLOY

Root directory: ./
Package manager: npm
Entry file: server.js
Node version: 20.x
Build command: npm install && npm run build
Start command: npm start
Output directory: vazio

Se a plataforma nao tiver Build/Start command, mantenha apenas:
- Entry file: server.js
- Package manager: npm
- Node 20.x

As variaveis de ambiente do banco devem ser cadastradas no painel.
Se DATABASE_URL nao for informada, o bootstrap monta automaticamente a URL a partir de DB_HOST, DB_PORT, DB_NAME, DB_USER e DB_PASS.
